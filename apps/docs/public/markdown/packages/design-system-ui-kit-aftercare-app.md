# Design System UI Kit: Aftercare App

## Purpose

This UI kit models the client-facing living aftercare page delivered by therapists.

## Composition

- `SegmentedToggle`
- `ProductCard`
- `Accordion`
- `Card`
- `StarMark`

## Notes

- Product visuals can be represented by placeholders during prototyping.
- Clinic name and therapist sign-off are intended for personalization.
