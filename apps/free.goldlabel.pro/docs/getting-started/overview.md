# Overview

NX is a modular application framework built on [Next.js](https://nextjs.org/) and [React](https://react.dev/).

It is designed as an opinionated template for shipping modern web apps quickly: routing, theming, content, state, animation, auth, and PWA concerns are already wired together.

The public repo is the framework itself. Clone it, configure a tenant, and use the NX runtime as a starting point for a branded Next.js application instead of assembling the stack from scratch.
