# python

Minimal Python app scaffold that participates in root Turbo tasks.

## Commands

- `pnpm --filter python dev`
- `pnpm --filter python build`
- `pnpm --filter python lint`
- `pnpm --filter python type-check`
- `pnpm --filter python test`

## Notes

- `lint` and `type-check` are intentionally dependency-free placeholders using Python stdlib compilation checks.
- Replace these scripts with tools like `ruff` and `mypy` when ready.
