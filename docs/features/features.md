# Features

- **Next.js 16** for SSR, SSG, API routes, and the App Router
- **TypeScript** for strict typing across the codebase
- **Material UI** for accessible, themeable UI components
- **Redux Toolkit** through Uberedux for flexible global state
- **Firebase** for authentication and backend integration
- **PWA support** for offline-first delivery
- **Multi-tenant architecture** for brand and client variations
- **Markdown-as-CMS** for file-based content management
- **REST API** exposed under `/api`
- **Design system** with reusable components, hooks, and icons
- **GSAP animation engine** through Flash and MovieClips
- **Shortcodes** for embedding React-driven content in Markdown
- **Rich media support** including maps, charts, images, SVG, and PDF
