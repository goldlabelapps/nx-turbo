# Design System UI Kit: Marketing Site

## Purpose

This UI kit models the public `leida.app` landing page for solo skin therapists.

## Composition

- `TopBar`
- `Button`
- `PriceTier`
- `Accordion`
- `StatCard`
- `RangeSlider`
- `Input`
- `Card`
- `Eyebrow`
- `Logo`
- `StarMark`

## Sections

- Sticky top bar
- Hero with CTA and preview
- How it works
- Cost calculator
- Social proof
- Pricing
- FAQ
- Waitlist
- Footer
