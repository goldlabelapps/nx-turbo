A pill-shaped text field on soft paper. Border warms to clay on focus.

```jsx
<Input label="Email" type="email" placeholder="you@clinic.com" />
<Input placeholder="Search products" />
<Input label="Email" error="That doesn't look right" />
```

- `label` - `hint`, `error` (error also tints the border clay)
- Passes through all native `<input>` props.
