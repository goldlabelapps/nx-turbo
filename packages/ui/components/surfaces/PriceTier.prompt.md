A membership pricing tier with a sparkle-bulleted feature list and optional "most popular" badge. `featured` gives the ink-filled hero treatment.

```jsx
<PriceTier
  name="Solo"
  price="£19" cadence="/mo"
  description="Everything one therapist needs."
  features={["Unlimited living pages", "Your clinic branding", "Client reminders"]}
  badge="Most popular" featured
/>
```
